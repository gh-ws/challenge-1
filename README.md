# amazing-app

Internal web application. Builds and deploys automatically on push to `main`.

## Development

```bash
npm install
npm run build
```

## CI

Builds are run automatically via GitHub Actions on every push and pull request. Build logs are uploaded as artifacts for debugging and retained for 1 day.

## Bugs & Support

Found a bug or have a question? [Open an issue](../../issues/new) and our bot will triage it.
